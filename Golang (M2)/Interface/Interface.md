Interface (Interface, Struct, Method)
- interface harus memiliki minimal 1 method
- method didalam interface tidak memiliki body (hanya definisi)
- struct adalah kumpulan variable
- method adalah function yang dimiliki oleh struct
- method harus memiliki receiver
