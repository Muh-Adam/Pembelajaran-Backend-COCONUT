Soal : Sistem Inventaris Dengan Manajemen User

1. Buat Database dengan Tabel
    Buat database dengan nama inventaris_db,lalu buat tabel-tabel di dalamnya denga nama:
    barang
    kategori
    supplier
    log_aktivitas
    Tentukan sendiri nama kolom untuk setiap tabel, lalu isi 10 data pada masing-masing tabel.

2. Buat User dan Atur Hak Akses
    Buat 3 user dengan nama admin, staff, dan auditor. Setiap user memiliki peran dan hak akses masing-masing.
    admin : Full akses (CREATE, SELECT, DELETE, INSERT, DLL) pada semua tabel.
    staf : SELECT dan INSERT pada tabel barang, kategori, dan supplier.
    auditor : SELECT pada tabel log_aktivitas
